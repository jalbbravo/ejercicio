# Ejercicio para Ingeniero DevOps

Microservicio escrito en Java/Sprint

### Pre-requisitos 📋
_Docker_

### Ejecución 🔧

```
docker-compose up
```

### Uso ⚙️

Acceso: http://localhost:8080/newaccount
Metodo: POST

Ejemplo:

```
{
  "accountUserName": "arojas8",
  "accountTypeID": 1,
  "accountBalance": 10000,
  "creationDateTime": "0000-00-00"
}
```
